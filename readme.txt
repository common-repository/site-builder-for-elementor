== Description ==

An intuitive solution to create custom header/footer for your site with [Elementor Page Builder](https://wordpress.org/plugins/elementor/) and site elements.

== Getting Started ==

= Minimum Requirements =

* WordPress 5.2 or greater
* Elementor Page Builder 2.8.0 or greater
* PHP version 5.6 or greater (PHP 7.2 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)

= Automatic installation =

To do an automatic install of this plugin, log in to your WordPress admin dashboard, navigate to the Plugins menu and click Add New.

In the search field, type “Site Builder for Elementor” and click Search Plugins. Once you’ve found this plugin, you can install it by simply clicking “Install Now”.

= Manual installation =

WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
